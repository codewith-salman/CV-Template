## 📖 About The Project

<img src="CV%20Template/image/screenshot.PNG" alt="CV Template Screenshot" width="100%" />

The **CV Template** project is a front-end resume design created using only **HTML and CSS**.  
It focuses on clean structure, modern styling, and responsive layout to present personal and professional information effectively.

This template is ideal for:
- Personal resume websites  
- Online CV hosting  
- Beginner front-end practice  
- Portfolio projects  

The layout is easy to customize and can be adapted for different professions and styles.

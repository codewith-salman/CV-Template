## 📖 About The Project

<img src="CV%20Template/image/screenshot.PNG" alt="CV Template Screenshot" width="100%" />

The **CV Template** is a front-end resume design created using **HTML and CSS**, focusing on clean structure, modern styling, and responsiveness.

This project was built to practice:
- Semantic HTML structure  
- Clean and professional layouts  
- Responsive design using pure CSS  

Perfect for:
- Personal resume websites  
- Online CV hosting  
- Beginner front-end practice  
- Portfolio showcase projects  

If you like this project, don’t forget to ⭐ the repository!

---

## ✨ Features

- Clean and modern CV layout  
- Fully responsive design  
- Well-structured sections for profile, skills, and experience  
- Lightweight and fast loading  
- Easy to customize for any profession  
- Minimal and professional UI
